叉积和点积

这里是向量运算模块的代码：

struct Point {
	double x, y;
	Point(double x=0,double y=0){}
	friend Point operator + (const Point &a, const Point &b) {return Point(a.x + b.x, a.y + b.y);}
	friend Point operator - (const Point &a, const Point &b) {return Point(a.x - b.x, a.y - b.y);}
	friend Point operator * (const Point &a, const double &b) {return Point(a.x * b, a.y * b);}
	double norm() {return sqr(x) + sqr(y);}
} p[maxn];
double det(Point a, Point b) {
    return a.x * b.y - a.y * b.x;
}
double dot(Point a, Point b) {
    return a.x * b.x + a.y * b.y;
}

判断线段是否相交

对于线段CD来说，判断AB是否在其两侧的方法就很简单了。如果CA→\overrightarrow {CA}CA和CB→\overrightarrow{CB}CB异号就证明AB在CD两侧。（参考叉积的几何意义）

另外对于线段共线（叉积为0）的情况就要特殊判断一下了，具体方法可以在文末参考文献中找到。
